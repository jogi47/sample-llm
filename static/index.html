Pyth API with Local LLM Integration

API Documentation

This API provides endpoints for managing items and generating text using local LLM models.

Use the interactive Swagger UI to explore and test the API:

Swagger UI ReDoc

LLM Endpoint Example

Generate text using the local LLM model with a POST request to /llm/generate:

curl -X 'POST' \
  'http://localhost:8000/llm/generate' \
  -H 'Content-Type: application/json' \
  -d '{
  "prompt": "What is the capital of France?",
  "max_length": 512,
  "temperature": 0.7
}'

Models Available

View information about available models with a GET request to /llm/info:

curl -X 'GET' 'http://localhost:8000/llm/info'
